# Media Files

Folder ini digunakan untuk menyimpan file audio dan video yang digunakan di website portfolio.

## File yang Diperlukan

### Audio
- `introduction.mp3` - File audio perkenalan (format MP3)
- `introduction.ogg` - File audio perkenalan (format OGG, alternatif)

### Video
- `introduction.mp4` - File video perkenalan (format MP4)
- `introduction.webm` - File video perkenalan (format WebM, alternatif)

## Cara Menambahkan File

1. **Untuk Audio:**
   - Rekam atau siapkan file audio perkenalan Anda
   - Simpan sebagai `introduction.mp3` dan `introduction.ogg` (opsional)
   - Letakkan file-file tersebut di folder `media/`

2. **Untuk Video:**
   - Rekam atau siapkan file video perkenalan Anda
   - Simpan sebagai `introduction.mp4` dan `introduction.webm` (opsional)
   - Letakkan file-file tersebut di folder `media/`

## Catatan

- Jika file tidak tersedia, browser akan menampilkan pesan bahwa format tidak didukung
- Untuk tugas, elemen HTML audio dan video sudah ada di `index.html`
- Anda dapat menggunakan file audio/video dari sumber lain atau membuat sendiri
