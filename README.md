## ğŸ–¼ï¸ Cara Mengganti Foto

Website ini sudah dilengkapi dengan placeholder images. Untuk mengganti foto:

1. **Foto Profil** (`images/profile/`):
   - Ganti `profile-photo.jpg` dengan foto profil Anda (ukuran disarankan: 800x800px)
   - Ganti `about-photo.jpg` dengan foto untuk section About (ukuran disarankan: 600x800px)

2. **Foto Proyek** (`images/projects/`):
   - Ganti `project-1.jpg` sampai `project-4.jpg` dengan screenshot proyek Anda
   - Ukuran disarankan: 800x600px atau rasio 4:3

3. **Foto Gallery** (`images/gallery/`):
   - Ganti `gallery-1.jpg` sampai `gallery-6.jpg` dengan foto dokumentasi Anda
   - Ukuran disarankan: 600x600px (square) atau sesuai kebutuhan

**Penting:** Pastikan nama file JPG sesuai dengan yang sudah ada di HTML agar foto langsung muncul!
